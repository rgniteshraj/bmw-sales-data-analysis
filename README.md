## 🧩 Project Overview
This project analyzes **BMW Global Sales Performance (2018–2025)**, focusing on vehicle sales, revenue generation, regional performance, and electric vehicle (EV) trends.

Using **Excel and Power BI**, the dataset was cleaned, transformed, and visualized to uncover key business insights and sales patterns.

---

## 🎯 Project Objectives

### 1. Analyze Sales Trends  
Study yearly and monthly sales trends to identify growth patterns.

### 2. Evaluate Model Performance  
Identify top-performing BMW models based on units sold.

### 3. Regional Analysis  
Compare revenue and sales across regions such as China, Europe, USA, and Rest of World.

### 4. Market Insights  
Analyze EV model performance and market share trends.

---

## 📂 Dataset Information

- Records: 3000+  
- Features: 13 columns  

**Includes:**
- Year, Month, Month Name  
- Region  
- Model  
- Units Sold  
- Revenue  
- EV Sales  
- Economic indicators  

---

## ❓ Problem Statement
- Analyze BMW’s global sales performance across years and regions  
- Identify top-performing models and revenue contributors  
- Study trends in electric vehicle adoption  
- Compare regional revenue distribution  
- Understand overall growth patterns in automotive sales  

---

## 🧾 Attribute Details

| Attribute Name | Data Type | Description |
|---|---|---|
| Year | Integer | Sales year (2018–2025) |
| Month | Integer | Month number (1–12) |
| Month Name | Text | Name of the month |
| Region | Category (Text) | Sales region (China, Europe, USA, Rest of World) |
| Model | Text | BMW vehicle model (e.g., X5, 3 Series) |
| Units Sold | Whole Number | Total number of vehicles sold |
| Revenue (€) | Currency | Total revenue generated from sales |
| EV Sales | Whole Number | Number of electric vehicles sold |
| GDP Growth (%) | Decimal Number | Economic growth indicator |
| Fuel Price Index | Decimal Number | Fuel price trend indicator |
| Total Sales Value | Currency | Calculated total sales amount (if used) |
| Price per Unit (€) | Currency | Average price per vehicle (calculated) |
| Sales Category | Text | Sales classification (High/Medium/Low) |
